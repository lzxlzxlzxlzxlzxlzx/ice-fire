## 2. 主菜单

主菜单是游戏的入口界面，包含以下功能：

```
┌─────────────────────────────────────┐
│          ICE & FIRE                 │
│       六边形战棋                    │
│                                     │
│    [ 开始游戏 ]                      │
│    [ 读取存档 ]                      │
│    [ 编辑器 ]                        │
│    [ 设置 ]                          │
│    [ 退出 ]                          │
│                                     │
│          v0.1                        │
└─────────────────────────────────────┘
```

- **开始游戏**：进入游戏设置界面
- **读取存档**：显示存档列表，选择后直接进入游戏（跳过设置）
- **编辑器**：进入编辑器选择菜单（仅开发者/管理员可见，或通过密码进入）
- **设置**：音效、画面、键位设置
- **退出**：关闭游戏

### 4.2 HUD 组件说明

| 组件 | 位置 | 内容 |
|------|------|------|
| 资源栏 | 顶部 | 当前所有资源的数量，每回合变化量 |
| 信息面板 | 右下 | 选中对象（单位/城市/单元格）的详细属性 |
| 操作栏 | 左下 | 当前选中单位的可用操作按钮 |
| 迷你地图 | 左下角 | 全地图缩略图，显示视野范围 |
| 回合信息 | 右上角 | 当前回合数、当前活动玩家 |
| 结束回合按钮 | 右下角 | 点击结束当前回合 |

### 4.3 弹窗/面板

| 弹窗 | 触发方式 | 内容 |
|------|---------|------|
| 城市面板 | 点击城市格 | 城市属性、等级、人口、HP、产出等 |
| 建筑面板 | 城市面板中点击"建筑" | 可建造的建筑列表、当前拥有的建筑 |
| 招募面板 | 城市面板中点击"招募" | 可招募的单位列表、消耗预览 |
| 暂停菜单 | 按 Esc | 继续游戏、保存、读取、设置、返回主菜单 |
| 战斗动画 | 攻击触发 | 战斗过程可视化（攻击方→防守方动画） |

## 6. 编辑器入口

编辑器在**主菜单**中作为独立入口（仅开发者可见），进入后选择编辑类型：

```
主菜单 → [编辑器] → ┌─ 地图编辑器
                    ├─ 单位编辑器
                    ├─ 建筑/改良编辑器
                    └─ 返回
```

编辑器详细设计见 **16-编辑器系统**。

---

## 7. 管理后台入口

管理后台是独立的 Web 页面（`/admin` 路由），不集成在游戏客户端内。

```
浏览器 → /admin → ┌─ 配置表管理（单位/词条/阵营等 CRUD）
                  ├─ AI 自对战控制台
                  └─ 数据导出/导入
```

---

## 8. 设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 编辑器在主菜单中 | 独立入口，不混入游戏流程 | 仅开发者使用，不对普通玩家暴露 |
| 管理后台为 Web 页面 | 与游戏客户端分离 | 方便在游戏外调整配置，不影响运行中的游戏 |
| 城市面板为弹窗 | 覆盖在地图画布上 | 保留地图背景参考，不切换全屏 |
| 先选中再操作 | 点击单位 → 显示可用操作 | 符合战棋类游戏操作习惯 |
