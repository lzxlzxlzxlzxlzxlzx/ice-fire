# 单位列表参考（基于原作）

> 本文档是 03-单位类型 的前置参考，仅列出原作中出现的军事单位。
> 后续设计 03 文档时将从这里筛选并定义游戏内兵种体系。

---

## 写在前面

所有军事单位按**兵种大类**组织，每个大类下再按**层级/定位**细分。标注各单位的：
- 使用阵营（参考 10-阵营概念）
- 原作定位与特点
- 游戏中可能的定位线索

## 一、步兵类

### 1.1 轻装步兵

| 单位 | 使用阵营 | 原作特点 | 备注 |
|------|---------|---------|------|
| 征召兵（Levies） | 全维斯特洛阵营 | 征召农民，装备简陋（草叉、镰刀），训练不足 | 基础炮灰单位 |
| 持矛兵（Spearmen） | 全阵营通用 | 长矛+盾牌，对抗骑兵最有效 | 维斯特洛军队骨干 |
| 投石兵（Slingers） | 多恩、夏日群岛 | 牧羊人常用，射程远但威力弱 | 极低成本的远程步兵 |
| 标枪兵（Javelin Men） | 多恩、山地氏族、野人 | 投掷标枪后近战，骚扰型 | 游击/轻步兵 |
| 野人掠袭者（Free Folk Raiders） | 野人 | 武器混杂（石斧、骨刀、长矛），无甲 | 低甲高机动 |
| 穴居人（Cave Dwellers） | 野人（霜牙山脉） | 苍白无眼，石器武器，惧光 | 极端特殊地形单位 |

### 1.2 正规步兵

| 单位 | 使用阵营 | 原作特点 | 备注 |
|------|---------|---------|------|

| 单位 | 使用阵营 | 原作特点 | 备注 |
|------|---------|---------|------|

## 六、雇佣兵团

> 这些是厄斯索斯的主要军事力量，各团有独立编制和风格。

| 名称 | 规模 | 特点 |
|------|------|------|
| 黄金团（Golden Company） | ~10,000人 | 最大最精锐，流亡维斯特洛骑士后代，有战象，信守合约 |
| 次子团（Second Sons） | ~500-800人 | 混编（维斯特洛+厄斯索斯），在丹妮与奴隶湾间摇摆 |
| 暴鸦团（Stormcrows） | ~500骑+数百步 | 达里奥·纳哈里斯曾任团长，骑兵为主 |
| 勇士团/血戏班（Brave Companions / Bloody Mummers） | ~500-1,000人 | 臭名昭著，暴行无数（赫伦堡、詹姆断手） |
| 风怒团（Windblown） | ~2,000人 | 骑兵为主，碎布王子统领 |
| 猫之团（Company of the Cat） | ~2,500人 | 历史上由泰兰·兰尼斯特组建，多为维斯特洛流亡者 |
| 长矛团（Long Lances） | 未知 | 骑兵团，近多斯拉克风格 |

## 十、单位数量级参考（原作设定）

| 兵力规模 | 代表 |
|---------|------|
| ~100,000+ | 多斯拉克卡拉萨（卓戈统一后）、异鬼军团 |
| ~40,000-60,000 | 维斯特洛大战役各方主力（河湾地、西境） |
| ~20,000-30,000 | 北境、风暴地、野人大军 |
| ~10,000-15,000 | 多恩、谷地（保留）、黄金团 |
| ~8,000 | 无垢者（丹妮莉丝购买） |
| ~2,000-6,000 | 金袍子君临守备队 |
| ~200-500 | 各城常备守军、佣兵团 |
| 7 | 御林铁卫 |
| 3 | 龙（丹妮莉丝时期） |
| ~200 | 巨人（野人盟友） |

> 以上兵力规模仅作为游戏经济/资源设计的参考基准，并非游戏内的实际数值。
