# 08 资源类型

## 1. 设计原则

- 每种资源有明确的获取途径和消耗用途
- 资源设计反映冰火世界的核心矛盾：**人力 vs 物资 vs 战略资源**
- 基础资源大众化，战略资源决定核心竞争力

## 3. 资源详细定义

### R01 金币 Gold

**原作对应**：金龙币（Gold Dragons），维斯特洛通用货币

**获取来源**（概念）：
- 城市产出（基础 + 建筑 + 改良）
- 贸易路线（港口城市、自由贸易城邦）
- 劫掠（铁群岛特色）
- 铁金库贷款（特殊事件，仅布拉弗斯）

**消耗用途**（概念）：
- 单位维护费
- 雇佣兵团签约
- 建筑建造
- 外交/贿赂
- 购买物资

**特殊规则**：金币无存储上限，负余额触发信用破产事件。

---

### R02 人力 Manpower

**原作对应**：人口、劳动力、征召兵源

**获取来源**（概念）：
- 城市产出（基础 + 建筑 + 改良）
- 占领/吞并城市
- 事件（难民涌入、战后归乡）

**消耗用途**（概念）：
- 招募单位
- 建造大型建筑
- 被占领城市强制征召

**特殊规则**：
- 人力上限 = 所有城市人口之和
- 人力不足时强行征召会降低城市忠诚度和人口增长
- 无垢者不消耗本地人力（在奴隶湾购买）

---

### R03 木材 Wood

**原作对应**：造船木材、攻城器械、建筑用料

**获取来源**（概念）：
- 城市改良产出（伐木场，需森林地貌）
- 进口贸易

**消耗用途**（概念）：
- 舰船建造
- 攻城器械建造
- 建筑建造
- 冬季取暖（可选，消耗木材减少人力损失）

---

### R04 石材 Stone

**原作对应**：城堡石料、凯岩城、鹰巢城

**获取来源**（概念）：
- 城市改良产出（采石场/矿井）
- 谷地特殊加成（山地多石）

**消耗用途**（概念）：
- 城墙建造
- 城堡/要塞建造
- 道路修建

---

### R05 钢铁 Steel

**原作对应**：武器盔甲用钢

**获取来源**（概念）：
- 城市改良产出（矿井）
- 西境最富

**消耗用途**（概念）：
- 高级兵种装备
- 舰船装甲（铁船）
- 武器升级

---

### R06 马匹 Horse

**原作对应**：战马、多斯拉克马

**获取来源**（概念）：
- 城市改良产出（牧场，需草原地形）
- 多斯拉克特殊加成

**消耗用途**（概念）：
- 骑兵单位训练
- 运输/补给线

**阵营差异**：
- 多斯拉克：马匹产出优势，马匹可替代部分金币
- 铁群岛：无法产马，骑兵极其稀少
- 谷地：优质牧场，良马产地

---

### R07 龙晶 Dragonglass（黑曜石）

**原作对应**：龙石岛特产黑曜石，杀异鬼利器

**稀有度**：稀有 —— 仅限特定单元格产出，数量有限

**获取来源**（概念）：
- 龙石岛（唯一稳定产地）
- 特定火山地形单元格
- 剧情事件/探索

**消耗用途**（概念）：
- 锻造龙晶武器（对异鬼特攻）
- 武装单位获得词条"龙晶武器"

> 龙晶对人族单位无效，战略价值集中在对异鬼战。

---

### R08 瓦雷利亚钢 Valyrian Steel

**原作对应**：传说级魔法武器——寒冰、长爪、守誓剑等

**稀有度**：传说级 —— 全局可能仅少量件数

**获取来源**（概念）：
- 开局世界散布（历史传世武器）
- 探索遗迹/特殊事件（极低概率）
- 无法生产（瓦雷利亚已毁灭）

**消耗用途**（概念）：
- 装备英雄/领袖单位（赋予词条"瓦雷利亚钢武器"）
- 效果：对异鬼秒杀，对人族攻击力大幅提升

> 瓦雷利亚钢不是常规"生产-消耗"型资源，更像战略宝藏，按件计数。

## 4. 资源层级体系

```
第一层（生存基础）        第二层（军事扩张）        第三层（核心竞争力）

   人力 ────────────→      钢铁 ──────────────→     瓦雷利亚钢（传说）
     │                      │
     │                      │
     ├── 招募单位 ────→  装备高级单位               马匹（战略）
     │                      │
     ↓                      ↓
   金币 ────────────→      木材                       龙晶（反异鬼命脉）
     │                    石材
     ├── 维护费
     ├── 买资源（贸易）
     ├── 雇佣兵
     └── 外交
```

| 层级 | 资源 | 控制失误后果 |
|------|------|-------------|
| 生存层 | 金币、人力 | 军队溃散、无人可征 |
| 扩张层 | 木材、石材、钢铁 | 无法扩张军备、防守薄弱 |
| 战略层 | 马匹、龙晶、瓦雷利亚钢 | 特定战术路线被封锁 |

## 6. 资源与阵营特色

| 阵营 | 资源倾向 | 特色机制 |
|------|---------|---------|
| NK 北境 | 人力尚可、钢铁中等 | 冬季人力恢复 -50%；木材充足 |
| RV 河间地 | 均衡、人力充足 | 冲积平原人口增长快；易被掠夺 |
| WL 西境 | 金币+钢铁之王 | 金矿产出 +50%，铁矿优质 |
| VL 谷地 | 石材+马匹 | 山地石材 +30%；良马产地 |
| SL 风暴地 | 木材+人力 | 森林多，民风彪悍 |
| RH 河湾地 | 人口增长最快 | 人口增长 +30%，人力上限最高 |
| DN 多恩 | 独特防御 | 沙漠战损耗低，人力恢复不受气候影响 |
| IR 铁群岛 | 掠夺经济 | 不自产木材/钢铁/马匹，全凭劫掠 |
| CR 王领 | 金币+龙晶 | 君临税收加成；龙石岛产龙晶 |
| NW 守夜人 | 依赖供给 | 人力恢复 -30%；马匹/钢铁靠南方 |
| WL 野人 | 全掠夺 | 无城市经济，人力来自部落人口 |
| WW 异鬼 | 无 | 不消耗资源，尸体即为"资源" |
| BV 布拉弗斯 | 金币+舰队 | 铁金库贷款机制；造船木材折扣 |
| VT 瓦兰提斯 | 贸易全能 | 贸易收入 +30%；可招募多种雇佣兵 |
| DH 多斯拉克 | 马匹之王 | 马匹产出 +50%，马可替代部分金币 |
| SB 奴隶湾 | 人力特殊 | 可通过购买奴隶补充人力（不消耗本地人口） |
| QT 魁尔斯 | 贸易+龙晶 | 贸易加成；男巫消耗龙晶研究魔法 |

## 7. 资源收入计算

城市产出公式见 **09-城市概念**，建筑和改良的具体产出值见 **10-建筑概念** 和 **11-单元格改良**。

```
总收入 = Σ 城市产出（基础 + 建筑 + 改良）+ 贸易收入 + 特殊收入 - 维护费
```

## 8. 设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 粮食→人力 | 改为人力 | 更直观体现人口效益；省去粮食生产和消耗的繁琐计算 |
| 权力资源 | 暂不设置 | 留待后续版本，当前聚焦经济/军事维度 |
| 马匹作为独立资源 | 新增 | 骑兵是维斯特洛军事核心，马匹地域分布差异大 |
| 龙晶 | 稀有战略资源 | 反异鬼唯一命脉，控制龙石岛=战略优势 |
| 瓦雷利亚钢 | 传说级资源 | 按件计数，每件有独立背景，不参与常规经济循环 |
| 无垢者不消耗本地人力 | 特殊规则 | 体现"买来即用"的价值 |
| 铁群岛无马 | 阵营限制 | 符合原著"铁民不骑马"的设定 |
