# 08 资源类型

## 1. 设计原则

- 每种资源有明确的获取途径和消耗用途
- 资源设计反映冰火世界的核心矛盾：**人力 vs 物资 vs 战略资源**
- 基础资源大众化，战略资源决定核心竞争力

## 3. 资源详细定义

### R01 金币 Gold

**原作对应**：金龙币（Gold Dragons），维斯特洛通用货币

**获取来源**：
- 城市税收（人力 × 税率 × 经济建筑加成）
- 金矿/矿产单元格（西境特色）
- 贸易路线（港口城市、自由贸易城邦）
- 劫掠（铁群岛特色）
- 铁金库贷款（特殊事件，仅布拉弗斯）

**消耗用途**：
- 单位维护费（每回合）
- 雇佣兵团签约费
- 建筑建造（部分消耗）
- 外交/贿赂
- 购买物资

| 消耗项 | 金额参考 [待平衡] | 说明 |
|--------|-----------------|------|
| 单位维护费 | 单位类型 × 2~10/回合 | 越高级越贵 |
| 雇佣兵团签约 | 500~2000 一次性 | 按团规模 |
| 基础建筑 | 50~200 | |
| 高级建筑 | 300~600 | |
| 贿赂/外交 | 50~500 视目标 | |

**金币无存储上限**（铁金库概念），但负余额会触发信用破产事件。

---

### R02 人力 Manpower

**原作对应**：人口、劳动力、征召兵源

**获取来源**：
- 城市人口增长（每回合自然增长 + 粮食/水源加成）
- 占领/吞并城市
- 事件（难民涌入、战后归乡）
- 特定建筑（军营、训练场）

**消耗用途**：
- 招募所有单位（每单位消耗一定人力）
- 建造建筑（大型建筑消耗人力）
- 被占领城市强制征召

| 消耗项 | 人力消耗 [待平衡] | 说明 |
|--------|-----------------|------|
| 征召兵 | 10 | 基础步兵 |
| 正规步兵 | 20 | 军士/剑士 |
| 骑士 | 30 | 重骑兵，精英 |
| 无垢者 | 5 | 买来即用，不消耗本地人力 |
| 基础建筑 | 5~15 | |
| 城堡/城墙 | 30~60 | 大型工程 |

**人力恢复**：
```
每回合人力恢复 = 城市数量 × 3 + Σ军营建筑加成
```

**人力上限** = 所有城市人口之和

> 无垢者不消耗本地人力（在奴隶湾购买），变相体现其价值——不依赖人口基础。
> 人力不足时，强行征召会降低城市忠诚度和人口增长。

---

### R03 木材 Wood

**原作对应**：造船木材、攻城器械、建筑用料

**获取来源**：
- 森林单元格砍伐
- 进口贸易
- 建筑（伐木场）

**消耗用途**：
- 舰船建造
- 攻城器械建造
- 建筑建造（非石质建筑）
- 冬季取暖（可选，消耗木材减少人力损失）

| 消耗项 | 木材量 [待平衡] |
|--------|----------------|
| 长船 | 60 |
| 战船 | 100 |
| 铁船 | 80 + 30铁 |
| 投石机 | 50 |
| 攻城锤 | 30 |
| 基础建筑 | 20~60 |

---

### R04 石材 Stone

**原作对应**：城堡石料、凯岩城、鹰巢城

**获取来源**：
- 采石场/山地单元格
- 谷地特殊加成（山地多石）
- 建筑（采石场）

**消耗用途**：
- 城墙建造（消耗大头）
- 城堡/要塞建造
- 道路修建（减少移动消耗）

| 消耗项 | 石料量 [待平衡] |
|--------|----------------|
| 城墙（每级） | 100 |
| 城堡 | 200 |
| 塔楼 | 50 |
| 道路（每格） | 10 |

---

### R05 钢铁 Steel

**原作对应**：武器盔甲用钢

**获取来源**：
- 铁矿/矿脉单元格（西境最富）
- 铁匠铺/炼钢厂建筑

**消耗用途**：
- 高级兵种装备（骑士、重装步兵需要钢铁才能训练）
- 舰船装甲（铁船）
- 武器升级（增加攻击力）

| 消耗项 | 钢铁量 [待平衡] | 说明 |
|--------|----------------|------|
| 骑士训练 | 15/单位 | 板甲+武器 |
| 重装剑士 | 10/单位 | |
| 无垢者 | 5/单位 | 刺剑+圆盾 |
| 铁船 | 30/艘 | 铁甲撞角 |
| 武器升级 | 20/级 | +攻击力 |

---

### R06 马匹 Horse

**原作对应**：战马（Destrier）、骑马战马（Palfrey）、多斯拉克马

**获取来源**：
- 牧场/草原单元格
- 建筑（马厩、牧场）
- 多斯拉克特殊：马匹产出 +50%（马是核心经济）

**消耗用途**：
- 骑兵单位训练（骑士、轻骑兵、弓骑兵）
- 运输/补给线（提升部队移动范围）
- 特殊消耗（劫掠后转移物资效率）

| 消耗项 | 马匹量 [待平衡] | 说明 |
|--------|-----------------|------|
| 骑士 | 3/单位 | 每位骑士1匹战马 |
| 轻骑兵 | 2/单位 | |
| 弓骑兵 | 2/单位 | |
| 运输加成 | 5/次 | 提升劫掠后撤效率 |

**阵营差异**：
- 多斯拉克：马匹产出 +50%，马匹可替代部分金币（以马易物）
- 铁群岛：无法产马，骑兵极其稀少（无骑士文化）
- 多恩：沙漠马耐热但数量少
- 谷地：优质牧场，良马产地

---

### R07 龙晶 Dragonglass（黑曜石）

**原作对应**：龙石岛特产黑曜石，杀异鬼利器

**稀有度**：★★★ 稀有 —— 仅限特定单元格产出，数量有限

**获取来源**：
- 龙石岛（唯一稳定的产出地）
- 特定火山地形单元格
- 剧情事件/探索

**消耗用途**：
- 锻造龙晶武器（对异鬼特攻）
- 武装单位获得词条"龙晶武器"（对异鬼伤害 ×3）
- 特殊反异鬼行动

| 消耗项 | 龙晶量 [待平衡] |
|--------|----------------|
| 龙晶匕首/箭头 | 1（武装一整单位） |
| 龙晶武器 | 3（精英单体，大幅提升对异鬼） |

> 龙晶对人族单位无效（普通黑曜石不破甲），战略价值集中在对异鬼战。
> 龙石岛是铁王座直辖地，控制龙石岛≈掌握反异鬼命脉。

---

### R08 瓦雷利亚钢 Valyrian Steel

**原作对应**：传说级魔法武器——寒冰、长爪、守誓剑等

**稀有度**：★★★★★ 传说级 —— 全局可能仅 2~5 件

**获取来源**：
- 开局世界散布（历史传世武器）
- 探索遗迹/特殊事件（极低概率）
- 无法生产（瓦雷利亚已毁灭）

**消耗用途**：
- 装备英雄/领袖单位（赋予词条"瓦雷利亚钢武器"）
- 效果：对异鬼秒杀，对人族攻击力 +30%，永不磨损

| 消耗 | 说明 |
|------|------|
| 1单位瓦雷利亚钢 | 足够锻造一把剑/一件武器 |

> 瓦雷利亚钢不是常规"生产-消耗"型资源，更像战略宝藏。
> 每一件瓦雷利亚钢武器都有独特名称和背景（寒冰、长爪、守誓剑等）。

## 4. 资源层级体系

```
第一层（生存基础）        第二层（军事扩张）        第三层（核心竞争力）

   人力 ────────────→      钢铁 ──────────────→     瓦雷利亚钢（传说）
     │                      │                          
     │                      │                          
     ├── 招募单位 ────→  装备高级单位               马匹（战略）
     │                      │                          
     ↓                      ↓                          
   金币 ────────────→      木材                       龙晶（反异鬼命脉）
     │                    石材
     ├── 维护费
     ├── 买资源（贸易）
     ├── 雇佣兵
     └── 外交
```

| 层级 | 资源 | 控制失误后果 |
|------|------|-------------|
| 生存层 | 金币、人力 | 军队溃散、无人可征 |
| 扩张层 | 木材、石材、钢铁 | 无法扩张军备、防守薄弱 |
| 战略层 | 马匹、龙晶、瓦雷利亚钢 | 特定战术路线被封锁 |

## 7. 资源收入计算

### 7.1 每回合收入框架

```
金币收入   = Σ城市税收(人力基数 × 税率 × 经济修正) + Σ单元格产出 + 贸易 - 维护费
人力恢复   = Σ城市人口增长 + 兵营/训练场加成
木材收入   = Σ森林单元格产出 + 伐木场
石材收入   = Σ采石场单元格产出
钢铁收入   = Σ铁矿单元格产出 + 炼钢厂
马匹收入   = Σ牧场/草原单元格产出 + 马厩
龙晶收入   = Σ龙石岛/火山单元格产出（极有限）
瓦雷利亚钢 = 不产出（仅开局分布+特殊事件）
```

## 8. 设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 粮食→人力 | 改为人力 | 更直观体现人口效益；省去粮食生产和消耗的繁琐计算 |
| 权力资源 | 暂不设置 | 留待后续版本，当前聚焦经济/军事维度 |
| 马匹作为独立资源 | 新增 | 骑兵是维斯特洛军事核心，马匹地域分布差异大 |
| 龙晶 | 稀有战略资源 | 反异鬼唯一命脉，控制龙石岛=战略优势 |
| 瓦雷利亚钢 | 传说级资源 | 按件计数，每件有独立背景，不参与常规经济循环 |
| 无垢者不消耗本地人力 | 特殊规则 | 体现"买来即用"的价值，机制上值得差异化 |
| 铁群岛无马 | 阵营限制 | 符合原著"铁民不骑马"的设定 |

## 9. 依赖关系

- 02-单元格类型：定义地形资源的单元格产出（本表 7.2）
- 03-单位类型：定义各单位的招募消耗（人力+资源）
- 09-城市/建筑概念：定义建筑如何影响资源产出/存储
- 06-回合概念：定义每回合资源结算时序与季节性修正
