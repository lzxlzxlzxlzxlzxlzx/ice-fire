## 2. 跨阵营角色索引

| 角色 | 出现阵营 | 说明 |
|------|---------|------|
| 劳勃·拜拉席恩 | 王领(叛乱后)、风暴地(叛乱时期) | 两个版本不同定位 |
| 史坦尼斯·拜拉席恩 | 王领(龙石岛)、风暴地 | 龙石岛vs风暴地继承权 |
| 丹妮莉丝·坦格利安 | 多斯拉克(龙妈骑战)、奴隶湾(龙妈解放) | 两个阶段不同玩法 |
| 席恩·葛雷乔伊 | 北境(史塔克养子)、铁群岛(铁种王子) | 身份撕裂的双版本 |
| 艾莉亚·史塔克 | 布拉弗斯(无面者) | 仅此版本 |

## 3. 设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 跨阵营角色 | 同一角色多版本 | 不同时期/身份玩法差异大，值得做 |
| 守夜人领袖 | 设杰奥、琼恩、山姆威尔三人 | 山姆武力弱但知识专精，对异鬼战有独特价值 |
| 布兰归属野人阵营 | 是 | 三眼乌鸦在塞外，机制特殊待定 |
| 河间地石心夫人 | 独立领袖 | 凯特琳死后复活的复仇形态，机制应完全不同 |
